CSS Selectors: Cheat Sheet

Selector Syntax Description
Universal Selector * {declaration;} Applies to all elements.
Type Selector h1, h2, h3 {decl;} Targets elements by type (h1-h3).
Class Selector p.note {decl;} Targets elements by class (p with a "note" class).
ID Selector #intro {decl;} Targets elements by ID (#intro).
Child Selector li>a {decl;} Targets elements that are direct children (all a children of li).
Descendant Selector p a {decl;} Targets all decscendant elements (all a inside of p).
Adjacent Sibling Selector h1+p {decl;} Targets the first element that is the next sibling (1st p after any h1).
General Sibling Selector h1~p {decl;} Targets all elements that are siblings (all p elemements after any h1).
